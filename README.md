# i3lock++
Lock your screen with original background !!

## Project description
The idea is simple : 
* take a screeshot of current window
* apply an effect to screenshot
* use i3lock to lock screen

## Dependencies
* *imagemagick*
* *i3lock*
* *scrot*

## List of effects
* blur
* pixel
* gray
* sepia
* charcoal
* paint
* spread
* vignette

## Usage
Very simple !

```
i3lock++ [EFFECT]
```

## Tips
Check dependencies and run i3lock++ 

```
./check-dep && ./i3lock++ [EFFECT]
```
